The Document Object Model DOM is a programming interface used to represent and interact with the structure of web documents like HTML. It connect HTML element to java script

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener on a parent element manages events triggered by its child elements, utilizing the event bubbling mechanism.
